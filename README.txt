== Description ==

Highly customizable carousel for Beaver Builder powered by [Slick Carousel](https://kenwheeler.github.io/slick/), 
for more information about the configuration options please visit the site.

The plugin's features are:

* Fully responsive
* The user can setup different breakpoints and configuration options for different screen sizes
* The default item information is: a title, image, content and a link
* User is able to use shortcodes to display an item or all the items in the carousel
* Possibility to use custom HTML code for the items
* Comes with a simple basic style
* Tested on Beaver Builder (pro version) Version 2.7.4.2

For questions or custom requirements please contact the plugin's author at zed.1985@gmail.com 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `nubocoder-bb-carousel` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The plugin will appear in the Beaver Builder modules
